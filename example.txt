1. Чому важливо вказувати user.name та user.email?

Git використовує ці дані, щоб підписувати коміти. Це дозволяє визначити автора змін і коректно відображати їх на GitHub.

2. Яка різниця між git pull і git fetch?

git fetch — завантажує нові зміни з сервера, але не зливає їх у твою гілку.

git pull — робить fetch + автоматично об’єднує зміни з твоєю поточною гілкою.

3. Як скасувати останній коміт, не втрачаючи зміни у файлах?
git reset --soft HEAD~1


Зміни залишаються в робочій директорії, коміт пропадає.
